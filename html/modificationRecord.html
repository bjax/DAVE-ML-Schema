modificationRecord

modificationRecord — To associate a reference single letter with a modification event

Content model

modificationRecord :  modID,  date,  [refID]
     author+
     description?
     extraDocRef*

Attributes

modID

A single letter used to identify all modified data associated with this modification record.

date

The date of the modification, in ISO 8601 (YYYY-MM-DD) format.

refID

(optional) A reference to a predefined document that describes the reason for this modification.

Description

A modificationRecord associates a single letter (such as modification "A") with modification author(s), address, and any optional external reference documents, in keeping with the AIAA draft standard.

Possible parents

fileHeader

Allowable children

author
description
extraDocRef
