1.1.2. Non-structural DTD and reference manual changes since version 2.0RC3

  • Noted that any math elements should include an xmlns attribute with the value of "http://www.w3.org/1998/Math/MathML". This is not a new requirement but helps parsers recognize the MathML namespace. Added this attribute to two examples that contained a math element.

  • Noted that the calculation element must contain exactly one math subelement.
