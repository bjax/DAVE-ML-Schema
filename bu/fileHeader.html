fileHeader

fileHeader — States source and purpose of file

Content model

fileHeader :  [name]
     author+
     (creationDate | fileCreationDate)
     fileVersion?
     description?
     reference*
     modificationRecord*
     provenance*

Attributes

name

(optional) The name of the file.

Description

The header element requires at least one author and a creation date; optional content includes version indicator, description, references, and modification records.

Possible parents

DAVEfunc

Allowable children

author
creationDate
fileCreationDate
fileVersion
description
reference
modificationRecord
provenance
