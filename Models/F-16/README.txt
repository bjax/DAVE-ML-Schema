       file for references).

       java -cp ~/Documents/Dev/java/xerces/xerces-2_0_1/build/classes sax.Counter -v F16_aero.xml

    
       To view using the sax tree viewer example:

       cd .. (up to DAVE directory)

       java javaxml2.SAXTreeViewer f16/F16_aero.xml
